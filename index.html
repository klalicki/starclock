The goal of this project was to design a novel means of timekeeping, that would be both accurately readable and interesting to look at. The Starclock uses the metaphor of a night sky to display the current time. Each of the three 'constellations' represents a different digit of the time; when the user clicks on the clock, the time appears in numerical form, but it can be read without numbers by simply counting the stars in each ring.

Built with JavaScript & p5.js
